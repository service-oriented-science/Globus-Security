Size Metrics :

The source statements is an indicator of the size of the system. Package volatility is a percentage measuring the fraction of packages that have been changed since previous release. Method volatility is a percentage measuring the fraction of methods that have been changed since previous release. Remember that this metric might be rather high since a method rename or move will imply 2 changes - one method is removed, and one method is added.
